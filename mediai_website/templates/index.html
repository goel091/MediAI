The Project

Built for Portfolio.
Designed for Real Use.

MediAI is a production-ready end-to-end data science project demonstrating real-world ML engineering — from raw data preprocessing through model training, SHAP explainability, to live web deployment.

The ML backend is written in Python using scikit-learn. The web layer is Flask + Vanilla JS — no heavy frameworks, just clean, fast, responsive code.

Python 3.11 Scikit-learn Flask SHAP Pandas / NumPy Joblib Docker Streamlit
🧬
Real ML Model
Trained Logistic Regression with StandardScaler, 5-fold CV, GridSearchCV tuning.
🔬
SHAP Explainability
Every prediction explained via SHAP values — see which symptoms drive the diagnosis.
📊
Full EDA

Docker Ready
